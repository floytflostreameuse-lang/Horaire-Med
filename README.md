# Luna — version GitHub Pages corrigée

Cette version est volontairement **à plat** : tous les fichiers, images et icônes doivent rester directement à la racine du dépôt.

## Installation sur GitHub

1. Supprime les anciens fichiers du dépôt ou crée un nouveau dépôt.
2. Décompresse le ZIP.
3. Dans GitHub : **Add file → Upload files**.
4. Sélectionne tout le contenu décompressé, y compris `index.html`, les `.svg`, les `.jpg`, les `.webp`, `app.js`, `styles.css`, `sw.js` et `manifest.webmanifest`.
5. Clique sur **Commit changes**.
6. Dans **Settings → Pages**, choisis **Deploy from a branch → main → /(root)**.
7. Attends 1 à 3 minutes, puis ouvre l’adresse GitHub Pages.

## Après une ancienne version

Le navigateur peut garder une ancienne version en cache. Sur la page du site :

- appuie sur `Ctrl + Shift + R`, ou
- ouvre les outils développeur (`F12`) → **Application** → **Storage** → **Clear site data**, puis recharge.
